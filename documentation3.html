NETWORKING MODULE

Networking stands for Interconnected systems that exchange data and resources. It enables communication between computers, mobile devices, and servers across local and global environments. This module covered the essentials of how networks operate, their architecture, protocols, and real-world applications.

Day 1

Introduction to Networking

  • Definition and importance of networking.
  • Types of networks: LAN, WAN, MAN, PAN.
  • Network characteristics: scalability, reliability, latency.
  • Basic components: router, switch, hub, modem, NIC.

Day 2

Network Topologies

  • Bus Topology: Single cable backbone, simple but limited.
  • Ring Topology: Circular data flow, predictable but vulnerable.
  • Star Topology: Central hub, scalable and easy to manage.
  • Mesh Topology: Multiple paths, high reliability.
  • Hybrid Topology: Combination of two or more topologies.

Day 3

OSI Model

  • OSI: Open Systems Interconnection model.
  • 7 Layers: Physical, Data Link, Network, Transport, Session, Presentation, Application.
  • Each layer has specific responsibilities in data transmission.
  • Helps standardize network communication and troubleshooting.

Day 4

Protocols & Services

  • UDP: User Datagram Protocol — fast, connectionless.
  • DNS: Domain Name System — resolves domain names to IP addresses.
  • DHCP: Dynamic Host Configuration Protocol — assigns IPs automatically.
  • Other protocols: TCP, FTP, HTTP, HTTPS.

Day 5

URL & Web Architecture

  • URL structure: protocol, domain, path, query string.
  • Client-server model: request-response cycle.
  • DNS lookup and IP resolution.
  • Secure communication using HTTPS and SSL/TLS.

Day 6

Network Design & Simulation

  • Designing small office/home networks.
  • Using simulation tools like Cisco Packet Tracer.
  • Assigning IPs, configuring routers and switches.
  • Testing connectivity and troubleshooting.
