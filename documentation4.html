Python & AIML Module

Overview

This module covered the fundamentals of Python programming and Artificial Intelligence & Machine Learning (AIML). The training was divided into 3 days of Python (NumPy, Pandas, Matplotlib) and 2 days of AIML. I learned how coding, data analysis, and machine learning come together to build intelligent systems.

Day 1 – Python Basics

  • Introduction to Python & IDEs (Jupyter, VS Code)
  • Variables, data types, operators
  • Control structures: if-else, loops
  • Functions & basic programs

Day 2 – NumPy Deep Dive

  • Creating 1D, 2D, and 3D arrays
  • Array indexing, slicing, reshaping
  • Mathematical operations: mean, median, std
  • Broadcasting and vectorized operations
  • Working with random numbers and linspace

Day 3 – Pandas for Data Analysis

  • Creating Series and DataFrames
  • Reading CSV and Excel files
  • Filtering rows and columns
  • Handling missing data
  • GroupBy, merge, sort, and pivot tables

Day 3 – Matplotlib Visualization

  • Line, bar, pie, scatter, histogram, box plots
  • Customizing plots: titles, labels, legends
  • Subplots and figure layout
  • Combining Pandas with Matplotlib

Day 4 – AIML Concepts

  • What is AI, ML, and Deep Learning?
  • Types of ML: Supervised, Unsupervised, Reinforcement
  • Training vs Testing data
  • Evaluation metrics: accuracy, precision, recall
  • Real-world applications: chatbots, vision, NLP

# Class for Book

class Book:

    def _init_(self, title, author, isbn):

        self.title = title

        self.author = author

        self.isbn = isbn

        self.available = True  # By default book is available



    def _str_(self):

        return f"{self.title} by {self.author} (ISBN: {self.isbn}) - {'Available' if self.available else 'Not Available'}">
